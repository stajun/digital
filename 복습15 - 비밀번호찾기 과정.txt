비밀번호 찾기(메일)
1- pom.xml 
  - 의존성추가(문서참조)

2- root-context.xml 
  - 메일 설정

- /main/login.jsp
  3- 비밀번호찾기 버튼 클릭 이벤트 등록
   4 - (테스트) 클릭이벤트 확인
   5 - prompt를 이용하여 비밀번호를 찾을 아이디를 입력 받음
   6 - (테스트) 입력받은 아이디 콘솔에 출력
   7 - ajax를 이용하여 서버에 데이터를 전송
      - data : 아이디
      - url : /find/pw
      - type : post
      22- 서버가 보내준 데이터가 fail이면 alert으로 '존재하지 않은 
        아이디'라고 알림
      23- 서버가 보내준 데이터가 success이면 alert으로 '비밀번호를
        등록한 메일로 전송했습니다'라고 알림

- HomeController
  8- /find/pw 담당 메소드 추가
    9- 화면에서 보낸 id를 받아옴(매개변수)
    10- (테스트) 받아온 id를 콘솔에 출력
    11- 서비스에게 id를 주면서 회원 정보를 요청
    12- (테스트) 요청한 회원정보 콘솔에 출력
    13- 회원정보가 없으면 "fail"을 리턴
    14- 비밀번호 생성
      이 때, 비밀번호는 1234로 함, 추후에 랜덤으로 수정
    15- 생성된 비밀번호를 암호화
    16- 암호화된 비밀번호를 회원정보(UserVo)에 수정(DB작업 아님)
    17- 서비스에게 회원정보를 주면서 회원정보 수정 요청(updateUser)
    20- 바뀐 비밀번호(1234)를 메일로 전송(문서 참조)
    21- 메일 전송이 완료되면 "success"를 리턴

- UserServiceImp
  18- updateUser 메소드 추가
    - 다오에게 회원정보를 주면서 회원 정보 업데이트를 요청
      (updateUser)

- UserMapper
  19- updateUser
    - id가 일치하는 회원의 비밀번호, 이메일, 나이, 홈페이지, 이름을
      수정하는 쿼리문 작성
